## 들어가면서

> 새로운 회사로 이직 후 오랜만에 개발 환경을 세팅했습니다.
> 오랜만에 하다 보니 옛 기억을 더듬어가며 이리저리 찾아보며 설정했는데,
> 그 때 나만을 위한 일종의 '매뉴얼'이 있으면 좋겠다는 생각이 들었습니다.
> 그래서 미래의 새 컴퓨터를 맞이할 나를 위한 개발 환경 세팅 매뉴얼을 작성해보려 합니다.

## 터미널

먼저 터미널부터 설치해줍니다. 저는 iterm 대신 [warp](https://www.warp.dev/)를 설치하려 합니다. <br/>
warp의 장점은 다음과 같습니다.

1. 명령 자동 완성 기능 제공
2. IDE와 유사한 편의 기능 제공
3. 예쁜(?) UI
4. Rust 기반 빠른 속도
5. zsh 기본 탑재
6. AI 기능 제공

그 다음 alias를 설정해줍니다.

```bash
vi ~/.zshrc
```

```bash
// i 입력 -> INSERT 모드 전환

// 아래 alias 추가
alias g=git
alias ..=cd ../

// esc -> :wq -> enter (편집 종료)
```

위 alias로 다음과 같이 터미널에서 사용할 수 있습니다. <br/>
자주 사용하는 명령어는 위와 같이 단축 설정을 하면 편리합니다.

```bash
// as-is
git clone [레포지토리 주소]
cd ..

// to-be
g clone [레포지토리 주소]
..
```

## 패키지

[homebrew](https://brew.sh/), [zoxide](https://github.com/ajeetdsouza/zoxide) 패키지를 설치합니다. <br/>
homebrew는
zoxide를 사용하면 경로를 기억해 어떤 depth도 한 번에 이동할 수 있습니다.

```bash
// 설치
brew install zoxide

// as-is
cd some/long/path/foo

// to-be
z foo
```

[nvm](https://github.com/nvm-sh/nvm), [yarn](https://yarnpkg.com/)을 추가로 설치합니다. <br/>
nvm을 설치하면 node, npm을 한 번에 관리할 수 있습니다.

```bash
brew install nvm
brew install yarn
```

다음 터미널 테마를 위해 [Powerlevel10k](https://github.com/romkatv/powerlevel10k)를 설치합니다. <br/>

```bash
// 차례대로 명령어 입력
git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ~/powerlevel10k
echo 'source ~/powerlevel10k/powerlevel10k.zsh-theme' >>~/.zshrc
```

설정 방법은 [링크](https://devocean.sk.com/blog/techBoardDetail.do?ID=165667&boardType=techBlog)를 참고하면 쉽게 따라할 수 있습니다.

## 유용한 프로그램

키보드 커스터마이징 프로그램인 [Karabiner-Elements](https://karabiner-elements.pqrs.org/)를 설치합니다. <br/>
맥 유저로써 한/영 전환이 꽤나 불편했는데 이 프로그램으로 윈도우와 유사하게 키를 적용할 수 있습니다.
저는 참고로 오른쪽 command 키를 한영 전환하는 키로 사용하고 있습니다.
<br/>

다음은 [Rectangle](https://rectangleapp.com/)입니다. <br/>
여러 프로그램을 한 화면에 보여주기 위해 창 크기를 수동으로 줄여서 맞출 때가 있는데, 단축키를 사용해 쉽게 화면 분할을 할 수 있습니다.
<br/>

세 번째는 [Giphy Capture](https://giphy.com/apps/giphycapture)입니다. <br/>
PR을 작성할 때 UI 동작 과정을 영상으로 캡쳐해 올리곤 하는데, 이 프로그램을 사용하면 gif(움짤), mp4(영상)를 선택해 캡처할 수 있습니다.
<br/>

마지막은 [Mail+ for Gmail](https://apps.apple.com/us/app/mail-for-gmail/id1216244845?mt=12)입니다. <br/>
업무용 메일로 보통 Gmail을 많이 사용할텐데 매번 브라우저로 들어가는 대신 한 번에 메일을 확인하고 알림을 받을 수 있습니다.

## IDE

저는 [Webstorm](https://www.jetbrains.com/ko-kr/webstorm/)을 사용하고 있습니다. <br/>
유료이긴 하지만 꽤나 편리한 기능들을 많이 제공하고 vscode만큼 다양한 플러그인이 있어 사용하고 있습니다. <br/>
이번에 프로젝트들을 설치하면서 알게 됐는데, 프로젝트 내 패키지의 종속성 취약점을 확인해 업데이트를 권장하는 기능도 알게 되었습니다.
<br/>

설치한 플러그인은 다음과 같습니다. <br/>

### [Power Mode II](https://plugins.jetbrains.com/plugin/8251-power-mode-ii)
IDE에 입력할 때 효과를 넣을 수 있어 개발하는 재미를 높일 수 있습니다.

### [Indent Rainbow](https://plugins.jetbrains.com/plugin/13308-indent-rainbow)
들여쓰기 영역을 컬러별로 보여주어 코드를 읽기 편하게 도와줍니다.

### [GitToolBox](https://plugins.jetbrains.com/plugin/7499-gittoolbox)
커밋 내역, 작성자, PR 이동 등 협업에 유용한 플러그인입니다.

### [Atom Material Icons](https://plugins.jetbrains.com/plugin/10044-atom-material-icons), [One Dark Theme](https://plugins.jetbrains.com/plugin/11938-one-dark-theme)
보다 예쁜 IDE를 위한 플러그인입니다. 저는 Atom One Dark 테마를 사용하고 있습니다.

### [Quick File Preview](https://plugins.jetbrains.com/plugin/12778-quick-file-preview)
파일을 더블 클릭하지 않고도 바로 볼 수 있게 해주는 플러그인입니다.

### [Translation](https://plugins.jetbrains.com/plugin/8579-translation)
IDE 내에서 바로 텍스트를 번역해 확인할 수 있습니다.
